== Description ==

Плагин для быстрой и удобной установки сервиса «СберЛид».
[СберЛид](https://sberlead.ru) - веб-сервис для повышения продаж на сайте за счет всплывающих поп-ап окон и виджетов для сбора лидов

= Польза для бизнеса: =
* До +20% к продажам с сайта
* Нет дополнительных расходов на маркетинг

= Преимущества: =
* 14 дней бесплатно
* Более 50 готовых шаблонов поп-ап окон под разный профиль бизнеса
* Удобный конструктор для создания индивидуального дизайна поп-апов
* Простое подключение без привлечения программистов

= Продукт состоит из нескольких модулей: =
* «Готовые виджеты» - мы сделали виджеты по наиболее популярным сценариям поведения посетителей. Правильно подобранный виджет может повысить вам конверсию до 20% и более
* «Конструктор виджетов» - легко настроить свой виджет под персональное предложение. С помощью конструктора вы можете создать свой виджет с уникальным дизайном и изображением
* «Лиды» - запустите виджеты сбора лидов и получайте заявки с контактными данными ваших клиентов (телефоны, e-mail адреса) прямо в личный кабинет СберЛид 
* «Слепые зоны» - узнайте на какие страницы не заходят посетители и повысьте скорость загрузки сайта.
* «Инструктор» - получайте персональные советы и рекомендации как улучшить сайт и повысить продажи.
* Размер ежемесячного платежа зависит от количества посещений вашего сайта в месяц.

= Возможности плагина =

== Installation ==

Процесс инсталляции плагина стандартен для WordPress.

1. Найти плагин в панели администратора вашего сайта:
Плагины->Добавить новый->Поле ввода "Поиск плагинов"-> Ввести "SberLead".
2. Установить найденный плагин.
3. Активировать плагин SberLead:
Плагины->Установленные->Активировать SberLead.
4. Перейти к настройке плагина->Вставить уникальный код с сайта sberlead.ru->Сохранить настройку
